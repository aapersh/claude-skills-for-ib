**Bulge-bracket-grade deal craft, codified into ten skills you can run on demand.**

Most deal work fails in the same places: a valuation range that is mechanically correct but commercially indefensible, a buyer list built without an acquisition rationale, a pitchbook polished into nothing, a CIM accepted at face value, an accretion model with synergies that quietly carry the entire thesis. These ten skills compress the workflow a strong analyst or associate would walk through on those problems — DCF / comps / precedents triangulation, peer normalization, control-premium logic, sponsor return math, buyer-universe segmentation, quality-of-earnings reading, accretion/dilution discipline, deal-process management — into structured workflows Claude can execute alongside you.

The skills split into two halves. **Analysis** builds the fact base, the valuation, and the deal thesis before you go to market. **Execution** carries the deal from pitch through process to close. Each skill includes the workflow a senior associate would walk through, the hypotheses to pressure-test, the quality checks before sending the deck upstairs, and the common failure modes to avoid.

**How to use them:** drop the folder into Claude, point it at the company, the comparable set, the CIM, or the live deal you are working on, and ask. No retainer. No 200-page IC memo template. No staffing meeting. Just rigor, on demand.

## Skills

### Analysis — build the fact base, the valuation, and the deal thesis

| Skill | What it does |
|---|---|
| Valuation Triangulator | Build a defensible valuation range across DCF, trading comps, precedent transactions, LBO, and premiums paid — and explain why the methodologies disagree |
| Precedent Transaction Screener | Build a screened, cycle-adjusted set of M&A precedents that actually inform pricing — not a long list of optically similar deals |
| Trading Comps Builder | Construct a clean trading comps universe with disciplined peer selection, calendarization, EBITDA adjustments, and outlier treatment |
| Company Teardown | Reverse-engineer a target's business model, unit economics, growth drivers, and risk profile from disclosed and inferred data |
| Market and Sector Brief | Produce the sector context any pitch, IC memo, or CIM must rest on — growth, cyclicality, M&A activity, capital markets backdrop |

### Execution — carry the deal from pitch through process to close

| Skill | What it does |
|---|---|
| Pitchbook Architect | Build a pitchbook with a real mandate logic — situation, why-us, valuation, buyer/investor angle, transaction roadmap — not a generic credentials deck |
| Buyer Universe Builder | Segment strategics and sponsors by acquisition rationale, capacity to pay, and process fit — and design the right outreach intensity |
| CIM Red-Flag Reviewer | Read a sell-side CIM the way a sharp buyer's MD reads it — surfacing quality-of-earnings risks, narrative gaps, and unsupported claims |
| Accretion / Dilution Tester | Run a defensible accretion/dilution analysis with the financing, synergy, and tax mechanics that actually move EPS |
| Deal Process Manager | Run an M&A or financing process end-to-end with the workstreams, timelines, and committee cadence that hold the deal together |
